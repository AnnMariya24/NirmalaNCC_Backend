# NirmalaNCC Backend

This is the backend API for the NirmalaNCC project, built using **Django** and **Django REST Framework**.

---

## 🛠 Features
* User authentication (NCC Cadets/Admin)
* Management of cadet records and attendance
* API endpoints for the frontend application
* [Add other features here, e.g., Image uploads for events]

## 🚀 Tech Stack
* **Language:** Python 3.x
* **Framework:** Django 5.x
* **Database:** SQLite (Development) / PostgreSQL (Production)
* **Tools:** Django REST Framework, WhiteNoise (for static files)
